# StyleGPT × Savana — AI Fashion Decision Engine

> *"Fashion shoppers don't buy products. They buy confidence."*

## What is this?
StyleGPT is an AI-powered styling layer built natively into Savana — a fashion commerce app. 
It solves three core problems: decision fatigue, high return rates, and low average order value 
by replacing search-and-filter with a conversational outfit engine.

## Live Prototype
🔗 [Try StyleGPT on Savana](https://style-savana-ai.lovable.app/)

**What to explore:**
- Tap "Create a farewell outfit under ₹4000" on the home screen
- Go to **Wardrobe** — see how StyleGPT styles around a piece you already own
- Try **Customize** — swap items, remove pieces, ask why
- Check **Me** — Fashion Memory + Smart Insights
- See **Try-On** and **Business Impact**

## The Problem
| Problem | Impact |
|---|---|
| Decision Fatigue | 40–60% conversion drop without styling context |
| High Return Rates | 28–35% of fashion orders returned industry-wide |
| Low AOV | Most sessions result in single-item purchases |

## The Solution — StyleGPT Features
| Feature | What it does |
|---|---|
| AI Chat Stylist | Describe occasion + budget → complete outfit in seconds |
| Wardrobe AI | Upload a piece you own → StyleGPT builds the rest |
| Customize Studio | Swap, remove, or ask "Why this?" in plain language |
| Personal Try-On | Preview the outfit on your photo — on-device, private |
| Fashion Memory | Learns your style over time. Every session smarter than the last |
| Bundle Commerce | Add entire AI-curated outfit to cart in one tap |

## Projected Business Impact
| Metric | Impact |
|---|---|
| Conversion Rate | +34% (AI-curated looks vs standard browse) |
| Average Order Value | +58% (bundle sessions vs single-item) |
| D30 Retention | +22% (StyleGPT users vs control) |
| Return Rate | −41% (AI-styled vs manually browsed) |
| Decision Time | −68% (session open to cart add) |
